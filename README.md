This project analyzes a student performance dataset to understand how different factors such as study hours, attendance,
parental education, and extracurricular activities impact academic results. The dataset was cleaned,
processed, and visualized using Python to extract meaningful insights.

Student_ID – Unique ID for each student
Age – Student age

Gender – Student gender

Class – Academic class/grade

Study_Hours_Per_Day – Daily study time

Attendance_Percentage – Attendance rate

Parental_Education – Education level of parents

Internet_Access – Internet availability (Yes/No)

Extracurricular_Activities – Participation status

Math_Score – Mathematics marks

Science_Score – Science marks

English_Score – English marks

Previous_Year_Score – Previous academic score

Final_Percentage – Final calculated percentage

Performance_Level – Performance category

Pass_Fail – Result status
